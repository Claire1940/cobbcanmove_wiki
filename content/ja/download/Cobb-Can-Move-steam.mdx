<Section icon={Target} title="初心者向けサバイバル＆操作ガイド">

**Cobb Can Move steam**の検索で序盤をマスターするには、このゲームが「ルールが変化する」サバイバルホラーであることを理解する必要があります。レベル1をプレイしたのと同じ方法でレベル2をプレイすることはできません。ゲームは、ダンジョンの奥深くに進むたびに、新しい一連の制約に適応することを強制します。

| 操作タイプ | キーボード入力 | ゲームパッド入力 | 機能 |
| :--- | :--- | :--- | :--- |
| **移動** | WASD / 矢印 | 左スティック | キャラクター移動 |
| **インタラクト** | E / スペース | ボタン南 | 拾う / 落とす / 使う |
| **投げる** | G / Q | ボタン西 | 持っているアイテムを投げる (石) |
| **メニュー** | Esc | スタート / メニュー | ポーズとレベル選択 |

<Steps>
  <Step num="1" title="ルールテキストを読む">
    レベルが開始されるたびに、テキストプロンプトが表示されます。これはゲーム内で最も重要な情報です。コブがあなたを見ているか、聞いているか、または現在飢えているかを教えてくれます。
  </Step>
  <Step num="2" title="石炭ルートを確立する">
    炉を見つけ、最も近い石炭の出現場所を見つけます。v1.6でマップ生成が更新され、石炭が炉のすぐ近くに出現する可能性が低くなり、より遠くまで移動する必要があります。
  </Step>
  <Step num="3" title="ブレーカーを管理する">
    電気が消えると、生存の可能性が低下します。ブレーカー（v1.3でより視認性の高いスプライトに更新）を見つけて、環境を明るく保ち、コブの視界を管理できるようにします。
  </Step>
  <Step num="4" title="死亡のヒントから学ぶ">
    死亡した場合でも、死亡画面をスキップしないでください。v1.4では、あなたが従わなかったルールを説明する特定のヒントが追加され、これは難しい感覚レベルを乗り越えるために不可欠です。
  </Step>
</Steps>

<Section icon={Shield} title="コブの能力と感覚ルール">

この挑戦の核心はコブの感覚にあります。固定されたAIを持つ従来のホラーモンスターとは異なり、コブの検出方法はアクティブなレベルのルールに基づいて変化します。これらのルールを理解することが、成功した実行と素早いジャンプスケアの違いです。

| ルール名 | コブの行動 | プレイヤーの対策 | 更新メモ |
| :--- | :--- | :--- | :--- |
| **コブは見ることができる** | 光の中でプレイヤーを検出 | 影の中に留まるか距離を保つ | v1.4/v1.5で弱体化 |
| **コブは聞くことができる** | 走る/投げる音を検出 | ゆっくり歩く (Shift) | 石で気をそらすことができる |
| **コブは匂いを嗅ぐことができる** | プレイヤーの位置を追跡 | 常に動き続ける | v1.7で大幅に弱体化 |
| **コブは届くことができる** | より長く、ゆっくりとした歩幅 | より大きな間隔を維持 | v1.6で調整 |
| **コブの複製** | 複数のコブが出現 | 追い詰められないようにする | v1.6でマージの修正 |

v1.7のアップデートでは、特に多くのプレイヤーが厳しすぎると感じていた「コブは匂いを嗅ぐことができる」ルールがターゲットになりました。コブは匂いを嗅ぐ頻度が減り、モンスターとの間に距離を置くことができれば、匂いを追跡する能力をより早く失います。

<Callout type="info" title="感覚の保証">
v1.4のアップデート以降、レベル2には少なくとも1つの感覚ルール（聴覚、嗅覚、視覚）が含まれることが保証されています。これは、後のレベルでより複雑な組み合わせに備えるためのチュートリアルとして機能します。
</Callout>

</Section>

<Section icon={Trophy} title="ストーリーモードの進行とエンドレスモード">

このゲームには2つの異なるプレイ方法があります。ストーリーモードは固定されたレベル進行（難易度の高いレベル7を含む）をたどり、エンドレスモードはベテランプレイヤーに手順的な挑戦を提供します。

| 特徴 | ストーリーモード | エンドレスモード |
| :--- | :--- | :--- |
| **進行** | レベル1からエンディングまで | 無限レベル |
| **難易度** | 固定カーブ | 時間とともにスケール |
| **レベル選択** | クリア後にアンロック | レベル3から開始 (v1.3) |
| **エンディング** | 最終タスクを含む | 固定されたエンディングなし |
| **マップタイプ** | 手作りのチャンク | プロシージャル生成 |

エンディングに苦戦しているプレイヤーのために、v1.6アップデートでは最終シーケンスで使用される「溶鉱炉」のヒットボックスが拡大されました。これにより、ストレスの高い追跡中に目標とインタラクトできなかったという理由だけでレベルに失敗するのを防ぎます。

<Callout type="info" title="リプレイ性">
一度ゲームをクリアすると、**レベル選択**メニューがアンロックされます。これは、ゲーム全体を再開することなく、後半のより難しいストーリーレベルで出現する特定のルール組み合わせを練習するのに最適です。
</Callout>

</Section>
