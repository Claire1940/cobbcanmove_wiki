<Section icon={Swords} title="プラットフォーム間の操作方法">

ウェブ版でもデスクトップ版でも、操作方法は一貫しています。開発者は、2026年の最新アップデートで、さまざまなキーボードレイアウトのサポートを含め、国際的なアクセシビリティのためにゲームを最適化しました。

<Steps>
  <Step num="1" title="移動とナビゲーション">
    移動には**WASD**または**矢印キー**を使用します。v1.6では、開発者が**AZERTY**レイアウトのサポートを追加し、ヨーロッパのプレイヤーが再マッピングなしでナビゲートしやすくなりました。
  </Step>
  <Step num="2" title="オブジェクトとの相互作用">
    石炭、石、バッテリーなどのアイテムを拾うには**E**または**左クリック**を使用します。アイテムを捨てるのは同じキーまたは専用のドロップボタンで行います。
  </Step>
  <Step num="3" title="投擲物の使用">
    石はあなたの主要な防御手段です。カーソルで狙いを定め、アクションキーで投げます。v1.7では、追跡中に石がコブをスタンさせないバグが修正されました。
  </Step>
  <Step num="4" title="環境の管理">
    視界を回復するには**ブレーカー**と、熱を維持するには**炉**と相互作用します。これらの相互作用は文脈依存であり、オブジェクトに直接隣接して立つ必要があります。
  </Step>
</Steps>

| アクション | キーボードマッピング | ゲームパッドマッピング (標準) |
| :--- | :--- | :--- |
| **移動** | WASD / 矢印 | 左スティック / D-パッド |
| **相互作用 / 拾う** | E / 左クリック | 南ボタン (A/クロス) |
| **アイテムを投げる** | 右クリック / Q | 右トリガー / 東ボタン |
| **ポーズメニュー** | Esc | スタート / オプション |
| **レベル選択** | マウス (メニューのみ) | ナビゲーションボタン |

<Callout type="success" title="ゲームパッドの利点">
多くのトッププレイヤーは、エンドレスモードでゲームパッドを使用することを好みます。アナログスティックは、**Cobb Can Hear**ルールがアクティブなときに非常に重要な、より繊細な「ステルスウォーキング」を可能にします。
</Callout>

<Section icon={Target} title="サバイバルメカニクスとルールシステム">

ゲームの中核は、その動的なルールシステムです。すべての**Cobb Can Move プラットフォーム**で、ゲームは各レベルの開始時にランダムなルールを生成します。画面上部に表示されるテキストに基づいて、すぐにプレイスタイルを適応させる必要があります。

<CardGrid cols={3}>
  <Card title="感覚ルール">
    - **Cobb Can See**: 視界が重要です。
    - **Cobb Can Hear**: 騒音は致命的です。
    - **Cobb Can Smell**: 近接が追跡されます。
  </Card>
  <Card title="移動ルール">
    - **Cobb Can Reach**: より大きな歩幅。
    - **Cobb Can Move**: 標準的な追跡。
    - **Duplicate**: 複数のコブが出現。
  </Card>
  <Card title="プレイヤーの制約">
    - **Freeze**: 光の近くに留まる。
    - **Starve**: ニンジンを集める。
    - **Darkness**: 視野が制限される。
  </Card>
</CardGrid>

生き残るためには、これらのルールを認識する能力が重要です。例えば、「Cobb Can Smell」というルールの場合、壁の裏に隠れても長くは助かりません。彼の追跡から逃れるためには、動き続ける必要があります。

| ルール名 | プレイヤーの戦略 | コブの行動変化 |
| :--- | :--- | :--- |
| **Cobb Can See** | 影に留まる / 視線（LOS）を遮る | 光の中にいるプレイヤーをターゲットにする |
| **Cobb Can Hear** | ゆっくり歩く / 石を使う | 大きな音を調査する |
| **Cobb Can Smell** | 常に移動する | プレイヤーに向かってゆっくりと漂う |
| **Duplicate** | 全方向を監視する | レベル5以降に2体以上のコブが出現する |
| **Reach** | 余分な距離を保つ | 大きく踏み込むようなステップを踏む |

<Callout type="info" title="v1.7 バランス変更">
2026年5月の最新アップデートでは、「Cobb Can Smell」ルールが弱体化されました。コブは匂いを嗅ぐ頻度が減り、近距離で追跡を見失った場合、しつこさが軽減されたため、逃げ出すチャンスが増えました。
</Callout>
