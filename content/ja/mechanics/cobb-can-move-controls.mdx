<Section icon={Map} title="キーボード操作の概要">
『Cobb Can Move』のキーボード操作は一般的で分かりやすく、方向移動と特定のアクションキーに焦点を当てています。このゲームは、AZERTYなどのさまざまなキーボードレイアウトとの互換性を向上させるためのアップデートが行われてきました。これらの基本的な操作を理解することが、ゲームの生存メカニズムをマスターするための第一歩です。

### 標準的なキーボード設定
以下は、一般的なキーボード操作をまとめた表です。特定のキー設定はカスタマイズ可能であるか、ゲームのバージョンやシステムによって若干異なる場合があります。

| アクション | デフォルトキー (QWERTY) | 機能 | アップデートノート |
| :-------------- | :------------------- | :----------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **上に移動** | W | プレイヤーキャラクターを上に移動させます。 | ナビゲーションやCobbを避けるために不可欠です。 |
| **下に移動** | S | プレイヤーキャラクターを下に移動させます。 | 退却やアイテム操作のための位置取りに不可欠です。 |
| **左に移動** | A | プレイヤーキャラクターを左に移動させます。 | 横方向の移動や回避に使用されます。 |
| **右に移動** | D | プレイヤーキャラクターを右に移動させます。 | 横方向の移動や回避に使用されます。 |
| **調べる/拾う** | E | アイテムを拾う、使用する、炉などのオブジェクトを操作する。 | v1.4では、ドロップ地点が壁の上で赤く表示されている場合でもアイテムの交換が可能になり、インベントリの管理がより容易になりました。 |
| **落とす/投げる** | Q | 現在持っているアイテムを落とす、または投げます。 | 石を投げてCobbを気絶させることができます（v1.6）。管理を誤るとパニックドロップにつながる可能性があります。 |
| **ライト切替** | F | プレイヤーの光源をオン/オフにします。 | 視認性とCobbの「見える」ルールの管理に不可欠です。 |
| **一時停止** | Esc | ゲームメニュー、設定へのアクセス、またはゲームの一時停止。 | プレイ中に一息ついたり、設定を調整したりするのに便利です。 |

<Callout type="warning" title="キーボードレイアウト">
v1.6アップデートでは、**AZERTYやその他の国際的なキーボードレイアウト**の改善が具体的に言及され、より幅広い層にとってゲームが快適になりました。QWERTY以外のレイアウトを使用している場合は、ゲーム内設定が最適化されているか確認してください。
</Callout>

### 一般的なゲームパッドのレイアウト
ゲームパッドの特定のデフォルト設定は詳細に公開されていませんが、多くのサバイバルホラーゲームは共通の形式に従っています。以下は、一般的なゲームパッド機能に基づいた予想される構成です。

| アクション | 予想されるゲームパッド入力 | 機能 | ゲームパッドユーザーの利点 |
| :-------------- | :--------------------- | :----------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **移動** | 左アナログスティック | 正確なナビゲーションのために、滑らかな360度の移動を提供します。 | デジタルのキーボード入力よりも微妙な移動制御が可能で、狭いスペースや回避に役立ちます。 |
| **調べる/拾う** | A / X ボタン | アクションの決定、アイテムの拾得、オブジェクトの使用。 | 素早い操作のための人間工学に基づいた配置。 |
| **落とす/投げる** | B / Circle ボタン | 保持しているアイテムを捨てる、または投げる。 | 手の位置を変えずに、投げられるアイテムを素早く処分したり戦略的に使用したりできます。 |
| **ライト切替** | Y / Triangle ボタン | プレイヤーの光源を起動または停止します。 | 特にCobbの「見える」ルールがアクティブな際に、光のレベルを管理するのに便利です。 |
| **一時停止** | Start ボタン | ゲームメニューを開きます。 | ほとんどのゲームで標準的であり、アクションを一時停止するための慣れ親しんだ方法です。 |
| **カメラ/照準** | 右アナログスティック | カメラビューの調整（該当する場合）または投擲アイテムの照準。 | 『Cobb Can Move』はトップダウン視点ですが、このスティックは注意を逸らすための照準や将来のアップデートで使用される可能性があります。 |
| **ダッシュ** | 左トリガー / L2 | 移動速度を上げます（利用可能な場合）。 | 素早い加速のために割り当てられる可能性がありますが、通常は速度よりも慎重な移動が優先されます。 |

### レスポンシブなゲームプレイの確保
*   **入力遅延を最小限に抑える:** HTML5ブラウザ版をプレイしている場合は、ブラウザが最新であることを確認し、システムリソースを消費する可能性のある不要なタブやアプリケーションを閉じてください。Windows版の場合は、ドライバが更新されていることを確認してください。
*   **ゲームパッドの互換性をテストする:** ゲームパッドを使用している場合は、ゲームを起動する前にオペレーティングシステムで正しく認識されていることを確認してください。一部のゲームパッドでは、最適なパフォーマンスを得るためにサードパーティ製のソフトウェアが必要になる場合があります。
*   **アイテム管理を練習する:** v1.4アップデートにより、ドロップ地点が壁の上で赤く表示されている場合でも、アイテムの交換がより寛容になりました。リソースを不適切な場所に慌てて落とさないよう、素早いアイテム交換を練習してください。
*   **死亡時のヒントから学ぶ:** v1.4以降、死亡時にヒントが表示されるようになりました。これらは多くの場合、よくある間違いや現在のルールセットに関連しており、次のプレイでの操作を改善するのに役立ちます。
*   **感度を調整する（可能な場合）:** 明記されていませんが、一部のゲームでは感度の調整が可能です。利用可能な場合は、これを微調整することで移動をより自然に感じられるようになります。
