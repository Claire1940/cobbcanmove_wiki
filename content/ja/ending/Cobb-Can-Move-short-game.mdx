<Section icon={Shield} title="サバイバルの基本：初心者ガイド">

**Cobb Can Move ショートゲーム**で最初の数回のプレイを生き残るには、素早い反射神経だけでは不十分です。環境をマスターする必要があります。各レベルは、コブの現在の能力を示すテキストプロンプトから始まります。このテキストを無視することは、新しいプレイヤーにとって最も一般的な死因です。

<Steps>
  <Step num="1" title="アクティブなルールを分析する">
    移動する前に、画面上部のテキストを読みます。「Cobb can see（コブは見ることができる）」と表示されている場合は、彼の視界に入らないようにします。「Cobb can smell（コブは匂いを嗅ぐことができる）」と表示されている場合は、匂いの痕跡を残さないように移動し続けます。
  </Step>
  <Step num="2" title="リソースの優先順位付け">
    主な目標は、炉を燃やし続けることです。石炭の塊を見つけて中央の炉に運びます。v1.6では、石炭の生成場所が炉から遠くに変更され、長距離移動の危険性が増しました。
  </Step>
  <Step num="3" title="ブレーカーと視認性">
    ライトがちらつくか消えた場合は、ブレーカーを探します。スプライトはv1.3で視認性を高めるために更新されました。ライトを点灯させておくことで、「Freeze（凍結）」ルールによる体力消耗を防ぎます。
  </Step>
  <Step num="4" title="防御的な石の戦術">
    常に石を携帯してください。v1.6およびv1.7のアップデート以降、石を直接当てるとコブをスタンさせることができます。これは追跡中にハードリセットできる唯一の方法です。
  </Step>
</Steps>

<Callout type="warning" title="死は教師である">
v1.4アップデートで**死亡ヒント**が追加されました。死んだ場合でも、すぐに画面をスキップしないでください。ゲームはどのように死んだかを分析し、あなたを捕らえたルールに特化したヒントを提供します。
</Callout>

<Section icon={Target} title="Cobb Can Move ショートゲーム：ルールとコブの能力">

**Cobb Can Move ショートゲーム**の核心は、ルール変更システムにあります。コブは単なるモンスターではなく、さまざまな感覚的および身体的ハンディキャップを持つ存在です。これらのルールのニュアンス、特に2026年のバランスパッチ後の変更点を理解することは、高レベルのプレイにとって不可欠です。

<CardGrid cols={2}>
  <Card title="感覚ルール">
    - **Cobb Can See（コブは見ることができる）**: 視覚はあなたの光のレベルに基づきます。影の中に留まってください。
    - **Cobb Can Hear（コブは聞こえる）**: 走ったり石を投げたりすると彼に警告します。静かに移動するには歩いてください。
    - **Cobb Can Smell（コブは匂いを嗅ぐことができる）**: あなたの経路を追跡します。v1.7で持続性が低下しました。
  </Card>
  <Card title="物理ルール">
    - **Cobb Can Reach（コブは届く）**: 彼は大きな歩幅で移動します。余分な距離を保ってください。
    - **Cobb Can Duplicate（コブは複製できる）**: 複数のコブが出現します。v1.6では、融合するのではなく互いを押し合うようになりました。
    - **Cobb Can Move（コブは移動できる）**: 加速が段階的に上昇する標準的な追跡モード。
  </Card>
</CardGrid>

<Callout type="success" title="v1.7でのルールバランス調整">
最新のアップデートでは、**匂い**のルールが弱体化されました。コブが匂いを嗅ぐ頻度が減り、彼があなたの位置を特定する前に痕跡を断ち切る時間が増えました。
</Callout>

| ルール名 | メカニック詳細 | 対抗戦略 |
| :--- | :--- | :--- |
| **See（見る）** | 光/距離に基づく検出 | 壁/柱の陰に隠れる |
| **Hear（聞く）** | スプリントまたは石の音で警告 | 歩行モードを切り替える (Shift) |
| **Smell（嗅ぐ）** | 見えない追跡 | 3秒以上立ち止まらない |
| **Reach（届く）** | 突進/歩幅の増加 | 石を使って距離を作る |
| **Duplicate（複製）** | マップ上の複数のエンティティ | すべての位置を追跡し、追い詰められないようにする |
| **Speed（速度）** | 時間経過による追跡加速 | 視線を遮って速度をリセットする |

</Section>

<Section icon={Trophy} title="ストーリーモードの進行とエンディング攻略">

**Cobb Can Move ショートゲーム**のストーリーモードは、クライマックスのフィナーレにつながる7つのレベルで構成されています。各レベルではより複雑なルール組み合わせが導入され、リソースを大量に消費する最終タスクでクライマックスを迎えます。

**レベルごとの内訳：**
- **レベル1-2**: 基本的な移動の紹介。レベル2では、検出メカニクスを教えるために感覚ルール（見る、聞く、嗅ぐ）が必ず登場します。
- **レベル3-6**: 「複製」および「到達」ルールの導入。マップはより大きく、よりプロシージャルになります。
- **レベル7**: 「試練」。v1.5では、コブの攻撃性の増加を補うために、このレベルに追加の石が追加されました。
- **エンディング**: 4つのバッテリーを配置し、**高炉**とインタラクトする必要があります。v1.6アップデートで高炉のヒットボックスが拡大され、プレッシャーの下でもクリックしやすくなりました。

<Callout type="warning" title="エンディングの難易度">
最終バッテリータスク中のコブの速度は、v1.5.2でわずかに上昇しました。コブが3タイル以内にいる場合は、最後のバッテリーを配置しようとしないでください。
</Callout>

| レベル | 主要目標 | 主要な危険 | アンロック |
| :--- | :--- | :--- | :--- |
| **レベル2** | 感覚を学ぶ | 確実な見る/聞く/嗅ぐ | レベル3 |
| **レベル5** | 複製導入 | 複数のコブ | 高度なルール |
| **レベル7** | バッテリー探し | 高い攻撃性 | フィナーレ |
| **エンディング** | 高炉 | 最大追跡速度 | レベル選択 |

</Section>

<Section icon={Sparkles} title="エンドレスモードとリプレイ戦略">

ストーリーをマスターすると、**Cobb Can Move ショートゲーム**は、適応能力を限界まで試すエンドレスモードを提供します。このモードはレベル3から始まり、導入段階をスキップして、プロシージャル生成の深い部分に直接突入します。

**エンドレスモードの特徴：**
- **シードによるランダム性**: 各ランはユニークなシードを使用し、アイテムの配置やマップのチャンクが常に異なることを保証します。
- **スケーリングするプレッシャー**: レベルが進むにつれて、コブはより速く速度を上げ、「飢餓」と「凍結」のタイマーはより攻撃的になります。
- **複製の積み重ね**: ストーリーモードとは異なり、エンドレスモードでは、後の段階で同時に2体以上のコブが出現することがあります。

<Callout type="tip" title="プロのリプレイヒント">
**レベルセレクト**メニュー（最初のクリア後にアンロック）を使用して、特にレベル7を練習してください。重いバッテリーを運びながらコブをかわす方法を学ぶのに最適な方法です。
</Callout>

| モード | 開始レベル | マップタイプ | 難易度スケーリング |
| :--- | :--- | :--- | :--- |
| **ストーリー** | レベル1 | 線形進行 | レベルごとに固定 |
| **エンドレス** | レベル3 | プロシージャルチャンク | 無限 / 時間ベース |
| **レベルセレクト** | 任意 | 練習モード | 静的 |

</Section>
