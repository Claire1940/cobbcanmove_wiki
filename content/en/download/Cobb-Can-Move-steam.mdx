Mastering the early game in **Cobb Can Move steam** searches requires understanding that the game is a "rule-changing" survival horror. You cannot play Level 2 the same way you played Level 1. The game forces you to adapt to a new set of constraints every time you descend deeper into the dungeon.

| Control Type | Keyboard Input | Gamepad Input | Function |
| :--- | :--- | :--- | :--- |
| **Movement** | WASD / Arrows | Left Joystick | Move character |
| **Interact** | E / Space | Button South | Pick up / Drop / Use |
| **Throw** | G / Q | Button West | Throw held item (Rocks) |
| **Menu** | Esc | Start / Menu | Pause and Level Select |

<Section icon={Shield} title="Cobb's Abilities and Sensory Rules">

The core of the challenge involves Cobb's senses. Unlike traditional horror monsters with fixed AI, Cobb's detection methods shift based on the active level rules. Understanding these rules is the difference between a successful run and a quick jump-scare.

| Rule Name | Cobb's Behavior | Player Countermeasure | Update Note |
| :--- | :--- | :--- | :--- |
| **Cobb Can See** | Detects player in light | Stay in shadows or keep distance | Nerfed in v1.4/v1.5 |
| **Cobb Can Hear** | Detects running/throwing | Walk slowly (Shift) | Rocks can distract him |
| **Cobb Can Smell** | Tracks player position | Keep moving constantly | Heavy nerf in v1.7 |
| **Cobb Can Reach** | Longer, slower steps | Maintain larger gap | Adjusted in v1.6 |
| **Cobb Duplicate** | Multiple Cobbs spawn | Avoid getting cornered | Fixed merging in v1.6 |

The v1.7 update specifically targeted the "Cobb can smell" rule, which many players found too punishing. Cobb now sniffs less frequently and loses your scent faster if you manage to put distance between you and the monster.

<Callout type="info" title="Sensory Guarantee">
Since the v1.4 update, Level 2 is guaranteed to include at least one sensory rule (Hear, Smell, or See). This serves as a tutorial to prepare you for the more complex combinations in later levels.
</Callout>

| Feature | Story Mode | Endless Mode |
