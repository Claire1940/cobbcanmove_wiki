### Diseño Típico de Gamepad
Si bien los enlaces predeterminados específicos para gamepads no se detallan en las fuentes disponibles, la mayoría de los juegos de terror de supervivencia siguen un esquema común. Aquí hay un desglose probable basado en la funcionalidad típica del gamepad:

| Acción          | Entrada Probable de Gamepad | Función                                                     | Beneficios para Usuarios de Gamepad                                                                                                                                                                                            |
| :-------------- | :--------------------- | :----------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Mover Personaje**| Joystick Analógico Izquierdo      | Proporciona un movimiento fluido de 360 grados para una navegación precisa.   | Ofrece un control de movimiento más matizado que las entradas digitales del teclado, útil para espacios reducidos y esquivar.                                                                                                                 |
| **Interactuar**    | Botón A / X           | Confirma acciones, recoge objetos, usa objetos.              | Colocación ergonómica para interacciones rápidas.                                                                                                                                                                             |

### Estrategias de Adaptación de Control
El conjunto de reglas dinámico del juego impacta directamente cómo debes utilizar tus controles.

  <Card title="Reglas Sensoriales">

  </Card>
  <Card title="Regla de Alcance">
    Si **"Cobb puede alcanzar"** está activo, Cobb da pasos más grandes. Esto significa que necesitas mantener una **mayor distancia** y ajustar tus patrones de movimiento para evitar su alcance extendido.
  </Card>
  <Card title="Uso de Objetos">
    Conocer tus teclas de **Interactuar (E/A)** y **Soltar/Lanzar (Q/B)** instintivamente es vital para gestionar recursos como carbón, rocas y zanahorias bajo presión.
  </Card>
  <Card title="Acciones de Emergencia">
    Ser capaz de **soltar rápidamente una roca** para aturdir a Cobb (actualización v1.6) o **alternar tu luz** para gestionar la visibilidad puede ser la diferencia entre la supervivencia y un final rápido.
  </Card>

### Asegurando una Jugabilidad Responsiva
*   **Minimiza el Retraso de Entrada:** Si juegas la versión de navegador HTML5, asegúrate de que tu navegador esté actualizado y cierra cualquier pestaña o aplicación innecesaria que pueda consumir recursos del sistema. Para la versión de Windows, asegúrate de que tus controladores estén actualizados.
*   **Prueba la Compatibilidad del Gamepad:** Si usas un gamepad, asegúrate de que sea reconocido correctamente por tu sistema operativo antes de iniciar el juego. Algunos gamepads pueden requerir software de terceros para un rendimiento óptimo.
*   **Practica la Gestión de Objetos:** La actualización v1.4 hizo que el intercambio de objetos fuera más indulgente, incluso si el punto de caída está sobre una pared y en rojo. Practica los intercambios rápidos de objetos para evitar soltar recursos por pánico en posiciones incómodas.
*   **Aprende de los Consejos de Muerte:** A partir de la v1.4, morir ahora muestra consejos. Estos a menudo se relacionan con errores comunes o el conjunto de reglas actual, ayudándote a refinar el uso de tus controles para la siguiente partida.
*   **Ajusta la Sensibilidad (si está disponible):** Aunque no se indica explícitamente, algunos juegos permiten ajustes de sensibilidad. Si está disponible, afinar esto puede hacer que el movimiento se sienta más natural.
